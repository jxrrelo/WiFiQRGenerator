# WiFiQRGenerator
Generate a QR Code of any WiFi network using valid credentials. Easily shareable, easily accessible.

## Installation:
```bash
pip install pyqrcode
```

## Usage:
```bash
python3 wifi_qr.py <Network SSID> <Password> <WPA/WPA2>
```
